# 电话拒接


### 使用方法
```typescript
// 收到电话来电事件后，调用该方法拒接电话
// 电话事件请参考监听事件列表
fcc.release()
```
<!-- **入参说明** -->
#### 入参说明

无

#### 出参说明

| **出参名称** | **数据类型** | **说明**                         |
| -------- | -------- | ------------------------------ |
| status   | number   | 调用接口返回结果的状态码，其含义参考融合通讯中台通用状态码。 |
| msg      | string   | 给开发者的文字提示信息                    |
| data     | object   | 返回的信息                          |
