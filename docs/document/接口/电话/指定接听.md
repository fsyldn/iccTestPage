# 指定接听


### 使用方法
```typescript
// 指定接听正在等待的用户
fcc.requestcall({callId: 'xxxxx',skillid:'xxx'}).then(res => {
  console.log('requestcall', res)
})
```
<!-- **入参说明** -->
#### 入参说明

| **参数名** | **数据类型** | **选取原则** |**说明** |
| ---------- | ------------ | ------------ | ------------------ |
| callId       | string       | 必选         | 从等待队列查询出来的通话标志  |
| skillId       | string       | 必选         | 技能id  |


#### 出参说明

| **出参名称** | **数据类型** | **说明**                         |
| -------- | -------- | ------------------------------ |
| status   | number   | 调用接口返回结果的状态码，其含义参考融合通讯中台通用状态码。 |
| msg      | string   | 给开发者的文字提示信息                    |
| data     | object   | 返回的信息                          |
