# 电话保持


### 使用方法
```typescript
// 座席对正在通话中或振铃中的语音呼叫进行保持操作
fcc.hold({ ishold: true })
```
<!-- **入参说明** -->
#### 入参说明

| **参数名** | **数据类型** | **选取原则** |**说明** |
| ---------- | ------------ | ------------ | ------------------ |
| ishold      | boolean       | 必选         | 保持：true 取消保持：false |

#### 出参说明

| **出参名称** | **数据类型** | **说明**                         |
| -------- | -------- | ------------------------------ |
| status   | number   | 调用接口返回结果的状态码，其含义参考融合通讯中台通用状态码。 |
| msg      | string   | 给开发者的文字提示信息                    |
| data     | object   | 返回的信息                          |
