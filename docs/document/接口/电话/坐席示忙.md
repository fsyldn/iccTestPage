### 使用方法
```typescript
// 座席显示空闲，回到接报队列
fcc.sayBusy()
```
<!-- **入参说明** -->
#### 入参说明
无

#### 出参说明

| **出参名称** | **数据类型** | **说明**                         |
| -------- | -------- | ------------------------------ |
| status   | number   | 调用接口返回结果的状态码，其含义参考融合通讯中台通用状态码。 |
| msg      | string   | 给开发者的文字提示信息                    |
| data     | string   | 返回的信息                          |
