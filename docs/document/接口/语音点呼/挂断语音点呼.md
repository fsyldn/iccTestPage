# 挂断语音点呼
::: warning 提示
<!-- warning -->
正在呼叫对方时，对方尚未接听可调用此方法取消呼叫。正在通话过程中通话双方均可调用此方法挂断通话。
:::

### 使用方法
```typescript
fcc.dialVoiceHangUp()
```
<!-- **入参说明** -->
#### 入参说明
无

#### 出参说明

| **出参名称** | **数据类型** | **说明**                         |
| -------- | -------- | ------------------------------ |
| status   | number   | 调用接口返回结果的状态码，其含义参考融合通讯中台通用状态码。 |
| msg      | string   | 给开发者的文字提示信息                    |
| data     | object   | 返回的信息                          |
