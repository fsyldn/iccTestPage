### 事件参数说明

| **参数名**  | **数据类型** | **说明** |
| ----------- | ------------ | --------------------------------------------- |
| data       | object       | 返回的数据信息callType: "0"    // 呼叫类型<br/>isVideo: "1"    // 是否为视频呼叫  0-否  1-是<br/>endType: "1"   // 结束类型 |
| eventType       | string       | 事件类型           |         
| resultCode       | number       | 返回结果代码             |         
| resultMsg       | string       | 返回结果信息             |
