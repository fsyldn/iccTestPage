## 使用方法

```typescript
// 被呼方触发
fcc.listen('OnDialVideoDropped', params => {
  // 处理开发者自己的业务逻辑
})
```

### 参数说明

| **出参名称** | **数据类型** | **说明**                                                     |
| ------------ | ------------ | ------------------------------------------------------------ |
| callId       | string       | 呼叫标识。 |
| type          | string       | 挂机类型1：正常挂机、2：异常挂机                               |
| msg         | string       | 异常挂机错误                                                   |
