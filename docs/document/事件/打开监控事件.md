## 使用方法

```typescript
// 当某个监控设备被打开时会收到此事件
fcc.listen('OnOpenMonitor', data => {
  // 处理开发者自己的业务逻辑
})
```


### 事件参数说明

| **参数名**  | **数据类型** | **说明** |
| ----------- | ------------ | --------------------------------------------- |
| data       | object       |  返回的数据信息：resId-监控设备的编号，callStatus-返回结果代码           |         
| eventType       | string       | 事件类型           |         
| resultCode       | number       | 返回结果代码             |         
| resultMsg       | string       | 返回结果信息             |
