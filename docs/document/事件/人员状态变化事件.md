# 人员状态变化事件

## 使用方法

```typescript
// 当人员状态发生变化，用户会收到此事件
fcc.listen('OnMemberStatusChange', params => {
  console.log(params)
  // 处理开发者自己的业务逻辑
})
```
<!-- **入参说明** -->


### 事件参数说明

| **参数名**  | **数据类型** | **说明**                       |
| ----------- | ------------ | --------------------------------------------- |
| icpCode       | string       | 融合通讯平台调度账号                         | 
| status       | string       | 1-在线 2-忙碌 3-离线                      | 
| extSysAccount      | string       | 外部账号                        | 
| deptId       | string       | 机构id                         | 
| userId       | string       | 用户id                         |
